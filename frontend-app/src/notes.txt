JSX(Javascript and xml[extendable markup language]):
----------------------------------------------------
->react will understand JSX code but browsers will not understand JSX code,it understands only js code (because every browser having Js engine)
->to convert JSX code to js we have next generation compiler called "babel JS"

-> JSX is similar to HTML , here it allows both JS and HTML code
